## 代码块

```javascript
// Mink — 用代码创造文字的艺术
function greet(name) {
    return `Hello, ${name}! Welcome to Mink.`;
}

console.log(greet('World'));
```

```python
# Python 示例
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        yield a
        a, b = b, a + b

print(list(fibonacci(10)))
```

## 表格

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Cmd+N` | 新建 | 创建新文件 |
| `Cmd+O` | 打开 | 打开文件 |
| `Cmd+S` | 保存 | 保存当前文件 |
| `Cmd+B` | 加粗 | 切换粗体 |
| `Cmd+I` | 斜体 | 切换斜体 |
| `Cmd+/` | 源码 | 切换源代码模式 |
