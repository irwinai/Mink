  • 所见即所得编辑

🚀 快速开始

npm install && npm start
写作是灵魂的呼吸。

快捷键

用键盘掌控一切

⌘N新建文件
⌘O打开文件
⌘S保存
⌘B加粗
⌘I斜体
⌘/源代码模式
⌘F查找替换
⌘\侧边栏
